Archiving in linux

tar - is an archiving program designed to store and extract files from an archive file known as tarfile to create it, move files you intend to bzip2 into a directory, for instance /home/user/test, and run the following command:

tar -cjf tarfilename.tar.bz2 teste/

It will create (c), bzip2 (j), the files (f) in that directory teste/ into the tarfile tarfilename.tar.bz2
